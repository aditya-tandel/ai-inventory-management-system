# AI Inventory Management System

This project demonstrates a simple AI-powered inventory management system built using Python and basic machine learning concepts.

## Features
- Inventory data loading
- Stock level classification
- Basic demand prediction using linear regression
- Exploratory data analysis

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn

## Purpose
Developed as part of an AI Project Internship to apply foundational AI and machine learning techniques to a real-world inventory management problem.

## Project Structure
- data/ : Sample inventory datasets
- notebooks/ : Exploratory data analysis
- src/ : Core application logic
- requirements.txt : Project dependencies

## How to Run
1. Clone the repository
2. Install dependencies using:
   pip install -r requirements.txt
3. Run the application:
   python src/main.py
